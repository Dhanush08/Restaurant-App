#### Features:

- The API returns the dish categories and the dishes in that category.
- The dish categories are be slidable.
- If a dish has addons(`addoncat`), then it will display the text **“Customizations available”**.
- Clicking on the `+` and `-` buttons will update the count accordingly and also on the cart icon on the top bar.
- Implemented maintainable and reusable coding standards.
- The tabs are hardcoded, their length do not change as per API.
